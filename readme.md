# SQL Queries Project

## Overview

This project contains solutions to SQL project assignment based on a commercial database schema. The objective is to demonstrate the use of Data Query Language (DQL) statements to retrieve and analyze data from a relational database using Microsoft SQL Server.

## Database Schema

The database consists of the following tables:

- Categories
- Customers
- Orders
- Order_Details
- Employees
- Suppliers
- Products

These tables are related through primary and foreign keys, allowing complex queries using joins and subqueries.

## Assignment Objectives

The queries in this project demonstrate the use of:

- SELECT statements
- WHERE clause filtering
- INNER JOIN and LEFT JOIN
- Aggregate functions (SUM, COUNT)
- GROUP BY and HAVING
- CASE expressions
- Subqueries
- String functions
- Date functions
- Sorting results with ORDER BY
